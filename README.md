본 문서는 크롬 확장프로그램 BeCalm: AI-Work Digest(이하 “서비스”)의 이용조건 및 개인정보 처리에 관한 사항을 규정합니다. 사용자는 서비스 설치·실행 및 “동의” 버튼 클릭(또는 이에 준하는 동작)을 통해 본 약관 및 개인정보 처리방침에 동의합니다.

개인정보 처리방침
1. 개인정보 처리의 원칙

회사는 개인정보 관련 법령을 준수하며, 보고서 생성에 필요한 최소한의 데이터만 수집·이용합니다. 사용자는 동의를 거부할 수 있으며, 동의하지 않을 경우 서비스의 핵심 기능(보고서 자동 생성 등)이 제한될 수 있습니다.

2. 수집하는 데이터
2.1 수집하는 데이터 (사용자 동의 시)

회사는 사용자가 동의한 경우, Daily Scrum 보고서 생성을 위해 다음 데이터를 수집·처리할 수 있습니다.

AI 도구 대화 내용(원문)

ChatGPT, Claude, Gemini 등에서의 질문과 응답(사용자가 기능을 활성화한 경우)

협업 도구 활동(메타데이터/요약)

Notion: 페이지 제목 등(연동/권한 승인 시)

Slack: 메시지 요약 등(연동/권한 승인 시)

Google Workspace 활동(메타데이터)

Docs/Sheets/Slides 문서 제목 및 편집 활동 메타데이터

예: 문서 식별자, 수정 이벤트 시각, 변경 여부 등

웹 브라우징 데이터

접속 Website URL(전체 URL)

인터랙션 메트릭(집계 지표)

클릭 횟수

스크롤 패턴(구간/빈도/방향 등)

타이핑 컨텍스트(입력 내용 제외: 입력 여부/분량/속도 등)

탭 전환 패턴

집중도 분포 등

2.2 수집하지 않는 데이터

회사는 아래 정보를 수집하지 않습니다.

비밀번호, 신용카드 정보 등 민감 정보

키 입력 내용(키로깅 없음)

복사/붙여넣기 한 내용

스크롤 좌표, 클릭 위치 등 정밀 좌표/포지션

사용자가 AI 대화나 문서 등에 민감정보를 직접 입력하는 경우, 그 입력된 원문이 보고서 생성 과정에 포함될 수 있으므로 민감정보 입력에 유의해야 합니다.

3. 개인정보 이용 목적

회사는 수집된 데이터를 다음 목적에만 사용합니다.

Daily Scrum 보고서 생성 및 제공

서비스 품질 개선을 위한 최소 범위의 오류·장애 대응 및 보안 로그 관리

회사는 광고 타게팅, 데이터 판매, 데이터 브로커 제공 등 보고서 생성과 무관한 목적으로 데이터를 이용하지 않습니다.

4. 처리의 법적 근거(동의)

회사는 사용자의 명시적 동의를 근거로 원문 데이터를 수집·이용·외부 전송합니다.

사용자는 동의를 거부할 수 있으며, 이 경우 핵심 기능이 제한될 수 있습니다.

사용자는 언제든지 설정에서 기능을 비활성화하거나 확장프로그램을 제거하여 동의를 철회할 수 있습니다.

5. 외부 전송 및 처리

회사는 보고서 생성(요약/구조화)을 위해 사용자가 동의한 원문 데이터를 외부 AI API(Anthropic Claude API)로 전송하여 처리할 수 있습니다.

전송 데이터: AI 대화 원문, 제목/활동 메타데이터, 전체 URL

전송 목적: Daily Scrum 보고서 생성

전송 시점: 보고서 생성 실행 또는 자동 생성 트리거 발생 시

전송 범위: 최소 범위

6. 보관 기간 및 파기

개인정보 처리 목적이 달성되면 지체 없이 파기합니다.

서비스 이용 중에는 보고서 히스토리 제공을 위해 보관할 수 있습니다.

계정 삭제 시 삭제일로부터 30일 이내 파기합니다.

전자 파일은 복구 불가능한 방식으로 삭제합니다.
