본 문서는 크롬 확장프로그램 **BeCalm: AI-Work Digest**(이하 “서비스”)의 이용조건 및 개인정보 처리에 관한 사항을 규정합니다. 사용자는 서비스 설치·실행 및 “동의” 버튼 클릭(또는 이에 준하는 동작)을 통해 본 약관 및 개인정보 처리방침에 동의합니다.

## 1. 용어 정의

1. **서비스**: 회사가 제공하는 크롬 확장프로그램 및 관련 서버/웹 기능 일체
2. **사용자**: 서비스를 설치·접속·이용하는 자
3. **원문 데이터**: 보고서 생성 전 단계에서 수집·처리되는 텍스트/메타데이터(예: AI 대화 원문, URL 등)
4. **보고서**: 서비스가 자동 생성·제공하는 Daily Scrum 요약 결과물
5. **처리위탁(수탁사)**: 회사로부터 개인정보 처리업무를 위탁받아 처리하는 외부 사업자(예: 클라우드, AI API 제공자)

## 2. 약관의 효력 및 변경

1. 본 약관은 사용자가 동의 후 서비스를 이용하는 시점부터 효력이 발생합니다.
2. 회사는 관련 법령을 위반하지 않는 범위에서 약관/방침을 변경할 수 있으며, 변경 시 서비스 내 공지, 웹페이지 공지, 이메일 안내 등 합리적인 방식으로 고지합니다.
3. 변경 이후에도 사용자가 서비스를 계속 이용하면 변경 약관에 동의한 것으로 봅니다.

- 사용자의 작업 기록을 기반으로 한 **정리/요약 결과물 제공**

# 개인정보 처리방침

## 10. 개인정보 처리의 원칙

회사는 개인정보 관련 법령을 준수하며, **보고서 생성에 필요한 최소한의 데이터만** 수집·이용합니다. 사용자는 동의를 거부할 수 있으며, 동의하지 않으면 서비스의 핵심 기능(보고서 자동 생성 등)이 제한될 수 있습니다.

## 11. 수집하는 데이터

### 11.1 수집하는 데이터 (사용자 동의 시)

회사는 사용자가 동의한 경우, Daily Scrum 보고서 생성을 위해 다음 데이터를 수집·처리할 수 있습니다.

1. **AI 도구 대화 내용(원문)**
- ChatGPT, Claude, Gemini 등에서의 질문과 응답(사용자가 해당 기능을 활성화한 경우)
1. **협업 도구 활동(메타데이터/요약)**
- Notion: 페이지 제목 등(연동/권한 승인 시)
- Slack: 메시지 요약 등(연동/권한 승인 시)
1. **Google Workspace 활동(메타데이터)**
- Docs/Sheets/Slides 문서 **제목** 및 **편집 활동 메타데이터**(예: 문서 식별자, 수정 이벤트 시각, 변경 여부 등)
1. **웹 브라우징 데이터**
- **접속 Website URL(전체 URL)**: 사용자가 방문한 페이지의 전체 URL
1. **인터랙션 메트릭(집계 지표)**
- 클릭 횟수, 스크롤 패턴(구간/빈도/방향 등), 타이핑 컨텍스트(입력 “내용” 제외: 입력 여부/분량/속도 등), 탭 전환 패턴, 집중도 분포 등

### 11.2 수집하지 않는 데이터

회사는 아래 정보를 수집하지 않습니다.

- 비밀번호, 신용카드 정보 등 민감 정보
- **키 입력 내용(키로깅 없음)**
- **복사/붙여넣기 한 “내용”**
- **스크롤 좌표, 클릭 위치 등 정밀 좌표/포지션**

> 사용자가 AI 대화/문서 등에 민감정보를 직접 입력하는 경우, 그 “입력된 원문”이 보고서 생성 과정에 포함될 수 있으므로 사용자는 민감정보 입력에 유의해야 합니다.

## 14. 외부 전송 및 처리(Claude API 사용)

회사는 보고서 생성(요약/구조화)을 위해 사용자가 동의한 원문 데이터를 외부 AI API(Anthropic Claude API)로 전송하여 처리할 수 있습니다.

- **전송되는 데이터(예시)**:
    - AI 도구 대화 원문(활성화 시)
    - 협업/워크스페이스 제목 및 활동 메타데이터(연동 시)
    - **전체 URL**
- **전송 목적**: Daily Scrum 보고서 생성을 위한 요약/분류/구조화 처리
- **전송 시점**: 사용자가 보고서 생성을 실행하거나 자동 생성 트리거가 발생하는 경우
- **전송 범위**: 보고서 생성에 필요한 **최소 범위**

사용자는 언제든지 설정에서 관련 기능을 비활성화하거나 확장프로그램을 제거하여 외부 전송을 중단할 수 있습니다.

# 데이터 수집 및 사용 동의 요약(사용자 고지문)

서비스는 사용자가 동의한 경우에 한하여 다음 데이터를 수집·이용하고, 보고서 생성을 위해 Claude API로 전송할 수 있습니다.

- **수집:** AI 대화 원문, 협업/워크스페이스 제목·활동 메타데이터, **전체 URL**, 집계형 인터랙션 메트릭
- **미수집:** 비밀번호/결제정보, 키로깅(키 입력 내용), 복사/붙여넣기 “내용”, 스크롤·클릭 정밀 좌표
- **목적:** Daily Scrum 보고서 생성(요약/구조화/정리)
- **국외이전:** 보고서 생성을 위해 미국(Anthropic 본사)로 데이터가 이전(처리위탁)될 수 있음
- **철회:** 언제든 설정에서 비활성화 또는 확장프로그램 제거로 중단 가능
