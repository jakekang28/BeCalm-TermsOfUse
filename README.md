🔒 개인정보 처리방침
회사는 사용자의 개인정보를 소중히 여기며, [Start] 버튼을 통해 사용자가 직접 제어하는 구간에 대해서만 최소한의 데이터를 수집합니다.

1. 개인정보 처리의 원칙 및 사용자 제어
수동 활성화: 본 서비스는 사용자가 확장프로그램 내 [Start] 버튼을 클릭했을 때만 데이터를 수집하며, [Stop] 버튼을 누르는 즉시 모든 수집을 중단합니다.

최소 수집: 보고서 생성에 필수적인 데이터 외에는 수집하거나 보관하지 않습니다.

2. 수집하는 데이터 항목
회사는 사용자가 [Start] 상태를 유지하는 동안 다음 데이터를 처리할 수 있습니다.

AI 도구 대화 원문: ChatGPT, Claude, Gemini 등에서의 질문과 응답 (기능 활성화 시)

협업 도구 활동 메타데이터: Notion 페이지 제목, Slack 메시지 요약, Google Workspace(Docs/Sheets/Slides) 문서 제목 및 편집 이벤트 시각 등

웹 브라우징 데이터: 접속 Website의 전체 URL

인터랙션 메트릭: 클릭 횟수, 스크롤 패턴, 타이핑 속도/분량 등 (입력 내용 제외)

※ 수집하지 않는 정보: 비밀번호, 결제 정보, 키 입력 내용(Text), 복사/붙여넣기 내용, 정밀 좌표 등.

3. 개인정보 이용 목적 및 외부 전송
목적: Daily Scrum 보고서 생성(요약/구조화) 및 서비스 품질 개선.

외부 전송(Claude API): 보고서 생성을 위해 수집된 데이터는 Anthropic(미국) 본사 서버로 전송되어 처리됩니다. 회사는 이를 광고 목적으로 사용하거나 제3자에게 판매하지 않습니다.

4. 보관 기간 및 파기
회사는 원칙적으로 계정 삭제 시까지 데이터를 보관하며, 사용자가 계정 삭제를 요청할 경우 30일 이내에 복구 불가능한 방식으로 영구 파기합니다.

5. 데이터 보안
전송 구간 암호화(TLS) 및 **저장 데이터 암호화(AES-GCM-256)**를 적용합니다.

Supabase의 행 단위 보안(RLS)을 통해 본인 외에는 데이터에 접근할 수 없도록 격리합니다.

📄 서비스 이용약관
6. 용어 정의
서비스: 회사가 제공하는 크롬 확장프로그램 및 관련 기능 일체.

보고서: 서비스가 자동 생성하여 제공하는 Daily Scrum 요약 결과물.

7. 서비스의 내용 및 권한
서비스는 사용자의 작업 맥락을 기반으로 업무 요약 및 로그 구조화 기능을 제공합니다.

기능 수행을 위해 필요한 브라우저 권한을 요청하며, 사용자는 언제든 권한을 철회할 수 있습니다.

8. 사용자 의무 및 면책 (베타/MVP)
사용자는 서비스를 비정상적인 방법으로 공격하거나 타인의 데이터에 접근해서는 안 됩니다.

본 서비스는 베타(MVP) 단계로, 요약 결과의 부정확성이나 일부 기능 오류가 발생할 수 있습니다. 회사는 고의나 중과실이 없는 한 이로 인한 손해에 대해 책임을 제한합니다.

9. 지식재산권 및 준거법
서비스의 모든 권리는 회사에 있으며, 사용자가 제공한 원문 데이터의 권리는 사용자에게 귀속됩니다.

본 약관은 대한민국 법령을 준거법으로 합니다.
